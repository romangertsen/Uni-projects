# Инвестиционная стратегия для клиента (Monte-Carlo + HRP + опционы)

Проект посвящён построению долгосрочной инвестиционной стратегии для частного клиента с консервативным риск-профилем.  
Основная задача — обеспечить устойчивое финансирование расходов семьи и спортивной школы при сохранении капитала и контролируемом уровне риска.


---

## Описание проекта

Клиент — частный инвестор с крупным капиталом и низкой склонностью к риску.  
Цель — сформировать инвестиционную стратегию, которая:

- покрывает обязательные расходы на горизонте 20 лет;
- минимизирует вероятность потери капитала;
- сохраняет возможность получения дополнительной доходности;
- остаётся устойчивой в стресс-сценариях.

Стратегия построена по принципу **двухслойного портфеля**:

- защитный слой — облигации и депозиты;
- активный слой — опционные стратегии (call-spread).

Такой подход позволяет отделить **задачу сохранения капитала** от **задачи получения доходности**.

---

## Используемые методы

В проекте применяются:

- моделирование Монте-Карло
- дисконтирование денежных потоков
- HRP-алгоритм (Hierarchical Risk Parity)
- моделирование лог-доходностей акций
- портфельная оптимизация
- опционные стратегии (call-spread)

---

## Финансовая модель защитного слоя

Для оценки необходимого капитала использовалась модель денежных потоков с учётом:

- инфляции
- доходности облигаций
- ежемесячных расходов
- горизонта планирования 20 лет

Метод Монте-Карло использовался для моделирования:

- инфляции
- доходности ОФЗ
- будущих расходов

Результат моделирования:

> В 90% сценариев для покрытия расходов в течение 20 лет достаточно **752 млн руб.**  

Это значение используется как минимальный объём защитного слоя.

---

## HRP-портфель

Для распределения активов использовался алгоритм **Hierarchical Risk Parity (HRP)**.

Причины выбора:

- устойчивость к шуму в данных
- отсутствие необходимости прогнозировать доходности
- естественная диверсификация риска
- отсутствие инверсии ковариационной матрицы

Алгоритм включает:

1. расчёт доходностей
2. корреляционную матрицу
3. преобразование корреляций в расстояния
4. иерархическую кластеризацию
5. квазидиагонализацию
6. рекурсивное распределение риска

---

## Моделирование цен акций

Для базовых активов использовалось моделирование Монте-Карло:

- лог-доходности предполагаются нормально распределёнными
- используется логнормальный процесс цен
- горизонт моделирования — 20 лет
- число сценариев — 10 000

Для каждой акции рассчитывались:

- средняя цена
- медиана
- 10-й перцентиль
- 90-й перцентиль

Это использовалось для оценки доходности опционной стратегии.

---

## Активный слой — опционная стратегия

Активная часть портфеля реализована через **call-spread**.

Даже при полном обнулении активной части стратегия остаётся устойчивой.

---

## Результаты моделирования стратегии

Моделирование на горизонте **20 лет** показало устойчивость стратегии.

Результаты:

| Показатель | Значение |
|----------|---------|
| Начальный капитал | 900 млн руб. |
| Доход облигационного слоя | 5 211 млн руб. |
| Доход опционной части | 2 582 млн руб. |
| Расходы | 1 287 млн руб. |
| Чистая прибыль | 6 506 млн руб. |
| Среднегодовая доходность | 13.6% |
| Общая доходность | 870.71% |


---

## Вывод

Разработанная стратегия:

- обеспечивает покрытие обязательных расходов;
- сохраняет капитал даже в стресс-сценариях;
- создаёт дополнительную доходность через опционы;
- соответствует консервативному риск-профилю клиента.

Разделение капитала на защитный и активный слои позволяет одновременно решать задачи финансовой безопасности и роста капитала.
