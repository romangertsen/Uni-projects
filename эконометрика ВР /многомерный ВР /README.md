# VAR и BVAR-моделирование доходностей акций (PG, MCD, MSFT)

## Описание проекта

В работе проводится моделирование совместной динамики доходностей акций трёх крупных американских компаний:

- Procter & Gamble (PG)
- McDonald’s (MCD)
- Microsoft (MSFT)

Все компании относятся к различным секторам экономики, но зависят от общих макроэкономических факторов. Это создаёт предпосылки для совместного моделирования временных рядов.

Цель проекта — исследовать взаимосвязь доходностей акций и построить многомерные модели временных рядов:

- VAR-модель
- байесовскую VAR (BVAR)

---

## Данные

Используются недельные цены закрытия акций:

- период: **30.11.2020 — 24.11.2025**
- источник: **Yahoo Finance**
- частота: **weekly**


Исходные ряды цен нестационарны, а доходности стационарны. 

---

## Предобработка

Для каждого временного ряда выполнены:

1. логарифмирование цен
2. переход к доходностям (первая разность логарифмов)
3. тесты стационарности:
   - ADF
   - KPSS
4. анализ ACF/PACF

Доходности PG, MCD и MSFT признаны стационарными.  

---

## VAR-модель

### Выбор лагов
Число лагов выбрано с помощью информационных критериев:

```
p = 1
```

### VAR(1)

Модель:

```
Y_t = A1 Y_{t-1} + u_t
```

Основные результаты:

- лаг MCD значим в уравнении PG
- лаг PG значим в уравнении MSFT
- большинство коэффициентов статистически незначимы 

---

## Диагностика VAR

Результаты тестов:

- модель **стабильна**
- **автокорреляции остатков нет**
- JB-тест отвергает нормальность
- ARCH-тест выявляет гетероскедастичность 

---

## Импульсные отклики (IRF)

Для доходности PG:

- шок MCD → краткосрочный положительный эффект
- шок MSFT → слабый быстро затухающий эффект 

---

## FEVD

Разложение дисперсии ошибки прогноза показывает:

- MCD и MSFT **слабо объясняют вариацию PG**
- основная динамика определяется собственными шоками PG

---

## BVAR-модель

Использован Minnesota prior.

Гиперпараметры:

```
κ0 = 0.2
κ1 = 0.5
κ2 = 2.0
κ3 = 10000
```

BVAR даёт:

- более сглаженные импульсные отклики
- меньшую амплитуду реакций
- аналогичные содержательные выводы  

---

## Сравнение VAR и BVAR

Ключевой результат:

- модели дают **очень похожие прогнозы**
- влияние MCD и MSFT на PG **небольшое и краткосрочное**
- BVAR обеспечивает **байесовское сглаживание коэффициентов**

Итог:
динамика доходности PG определяется прежде всего собственными шоками.
