# Sentiment Analysis на филиппинском языке (Binary Text Classification)

## Описание проекта

В проекте решается задача **определения тональности текста (sentiment analysis)** на филиппинском языке.  
Модель должна классифицировать текст как:

- положительный (positive)
- отрицательный (negative)

Анализ тональности текста является важной задачей обработки естественного языка (NLP) и широко используется в:

- анализе отзывов пользователей
- мониторинге социальных сетей
- анализе клиентского опыта
- маркетинговой аналитике

Особенность проекта — работа с **текстами на филиппинском языке**, для которого существует меньше готовых NLP-инструментов по сравнению с английским.

---

## Постановка задачи

Задача формулируется как **бинарная классификация текста**.

Дано:
- текстовое сообщение
- метка тональности

Необходимо:
- преобразовать текст в числовое представление
- обучить модель классификации
- оценить качество предсказаний

---

## Данные

Датасет содержит:

- текстовые сообщения на филиппинском языке
- бинарную целевую переменную sentiment

Перед обучением модели были выполнены стандартные шаги обработки текстовых данных.

---

## Предобработка текста

В проекте использованы базовые методы NLP-предобработки:

- приведение текста к нижнему регистру
- удаление пунктуации
- удаление стоп-слов
- векторизация текста

Для преобразования текста в признаки использована модель **TF-IDF**.

---

## Моделирование

Были обучены модели линейной классификации текста.

Pipeline включает:

```
TF-IDF → классификатор
```

Использовались модели:

- Logistic Regression
- Linear SVM

---

## Метрики качества

Качество модели оценивалось с помощью ROC-AUC


---

## Результаты

Модель успешно научилась определять тональность текста на филиппинском языке.


Лучшие результаты показала линейная SVM-модель.

ROC-AUC : 0.783
