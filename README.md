# Университетские проекты 

В данном разделе представлены учебные и исследовательские проекты, выполненные в рамках обучения на экономическом факультете МГУ.  
Проекты включают задачи из эконометрики, машинного обучения, финансового моделирования и анализа сетевых данных.  
Основные языки программирования — **Python** и **R**.

---

| Проект | Описание | Навыки и инструменты |
|--------|----------|----------------------|
| [Сетевые модели (графы)](./сетевые%20модели%20(графы))| Анализ графовых структур и социальных сетей, исследование связности и центральности узлов | R, igraph, анализ графов, визуализация сетей |
| [Финансовое моделирование](./финансы) | Моделирование финансовых показателей и анализ инвестиционных решений | R, временные ряды, статистическое моделирование, финансовый анализ |
| [Эконометрика временных рядов (VAR/BVAR)](./эконометрика%20ВР) | Построение и сравнение VAR и Bayesian VAR моделей для макроэкономических данных | R, econometrics, VAR, BVAR, прогнозирование временных рядов |
| [Машинное обучение](./машинное%20обучение) | Бинарная классификация текстов на филиппинском языке по тональности | python, NLP, sklearn, TF-IDF, классификация текстов |
