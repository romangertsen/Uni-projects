# Университетские проекты 

В этом разделе представлены учебные и исследовательские проекты, выполненные в рамках обучения на экономическом факультете МГУ.  
Проекты включают задачи из эконометрики, машинного обучения, финансового моделирования и анализа сетевых данных.  
Основные языки программирования — **Python** и **R**.

---

| Проект | Описание | Навыки и инструменты |
|--------|----------|----------------------|
| [Сетевые модели (графы)](./сетевые_модели_графы)| Анализ графовых структур и социальных сетей, исследование связности и центральности узлов | R, igraph, анализ графов, визуализация сетей |
| [Финансовое моделирование](./финансы/финасовая_стратегия) | Моделирование финансовых показателей и анализ инвестиционных решений | Python, временные ряды, статистическое моделирование, финансовый анализ |
| [Эконометрика временных рядов (VAR/BVAR)](./эконометрика_ВР) | Построение и сравнение VAR и Bayesian VAR моделей для макроэкономических данных | R, econometrics, VAR, BVAR, прогнозирование временных рядов |
| [Прогноз сентимента](./машинное_обучение/прогноз_сентимента_на_филиппинском) | Бинарная классификация текстов на филиппинском языке по тональности | python, NLP, sklearn, TF-IDF, классификация текстов |
| [Прогнозирование доходности золота](./финансы/доходность_золота) | Построение ансамбля моделей для прогнозирования log-return золота и разработка торговых стратегий с оптимизацией портфеля по Марковицу | Python, временные ряды, feature engineering, IC, Sharpe ratio, mean-variance оптимизация, ансамбли моделей |
| [Financial Machine Learning для ETF](./курсовая_финансы) | Курсовая работа по применению методов Financial Machine Learning для фильтрации momentum-сигналов и управления размером позиции на рынке ETF. Реализованы CUSUM-фильтр, Triple Barrier Method, Fractional Differentiation, Meta-Labeling, PurgedKFold и walk-forward backtesting | Python, Financial Machine Learning, ETF, CUSUM, Triple Barrier Method, Meta-Labeling, LightGBM, Random Forest, portfolio management |
